# Java
A collection of Java projects, which are focused on object-oriented programming, file handling, arrays, data structures, user input, simulations, and basic graph/path-finding logic.

## Language

The application interface and code comments are written in Slovenian.

---
### 1. Matrix simulation

A program that generates and updates a two-dimensional matrix based on predefined neighbour rules. The project demonstrates working with 2D arrays, loops, conditions, random values, and user input.


### 2. Flight Route Finder

A Java program that reads flight connections from a file and finds possible routes between selected airports. It supports direct flights and routes with up to two transfers using structured data processing and basic path-finding logic.


### 3. Package Locker Simulation

An object-oriented Java application that simulates a package locker system. It includes classes for packages, lockers, and the locker system, with functionality for delivering packages, checking available lockers, and collecting packages based on recipient and locker ID.
